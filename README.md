### Installation
Grab the latest Inyector NuGet package and install in your solution. https://www.nuget.org/packages/Inyector/
```sh
PM > Install-Package Inyector 
NET CLI - dotnet add package Inyector 
paket add Inyector --version 0.1.1	
```
